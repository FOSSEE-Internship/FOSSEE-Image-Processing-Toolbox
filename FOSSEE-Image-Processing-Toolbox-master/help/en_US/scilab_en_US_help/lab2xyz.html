lab2xyz

This function is used to Convert CIE 1976 L*a*b* to CIE 1931 XYZ.

Calling Sequence

xyz = lab2xyz(lab)
xyz = lab2xyz(lab,Name,Value)

Parameters

lab:

Color values to convert, specified as a P-by-3 matrix of color values (one color per row), an M-by-N-by-3 image array, or an M-by-N-by-3-by-F image stack.

Whitepoint:

Reference white point, specified as a 1-by-3 vector or one of the CIE standard illuminants.eg,'a','c','e','d50','d55','d65','icc'.

xyz :

Converted color values, returned as an array the same shape and type as the input.

Description

xyz = lab2xyz(lab) converts CIE 1976 L*a*b* values to CIE 1931 XYZ values.xyz = lab2xyz(lab,Name,Value) specifies additional options with one or more Name,Value pair arguments.

Examples

tr1 = lab2xyz([50 10 -5]);
tr1
tr2 = lab2xyz([50 10 -5],'WhitePoint','d50');
tr2
