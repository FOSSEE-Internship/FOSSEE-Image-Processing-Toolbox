rgb2ntsc

This function is used to convert the range of rgb values to the range ntsc values.

Calling Sequence

image1 = imread(img);
image2 = rgb2ntsc(image1);

Parameters

image1:

image matrix of the source image.

image2:

image matrix of the resultant image.

Description

This function takes an rgb image and transforms the channels of the image in accordance with the ntsc values.

Examples

i = imread("lena.jpeg");
rr = rgb2ntsc(i);
imshow(rr(:,:,1));
imshow(rr(:,:,2));
imshow(rr(:,:,3));
