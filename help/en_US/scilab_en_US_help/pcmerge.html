pcmerge

Merge two point clouds

Calling Sequence

pointCloud = pcmerge(pointCloud1,pointCloud2,girdSize)

Parameters

pointCloud1 :

First point cloud

pointCloud2 :

Second point cloud

gridSize :

Parameter specifying the size of the grid used to filter the result

pointCloud :

Resultant point cloud after merging the two input point clouds

Description

Merges the two input point clouds and applies a gird filter

Examples

s1 = pcread('data/cycloidal.ply')
s2 = pcread('data/teapot.ply')
s = pcmerge(s1,s2,.1)
pcshow(s.Location)
