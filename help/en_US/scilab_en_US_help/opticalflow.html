opticalflow

Calculates optical flow.

Calling Sequence

opticalflow(vid)

Parameters

vid :

a video

Description

opticalflow(vid) returns an opticalflow video and it will be written to 'out.avi' file (In the current working directory).

Examples

opticalflow('ball.mp4');
opticalflow('vtest.avi');
