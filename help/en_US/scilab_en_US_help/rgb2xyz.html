rgb2xyz

This function is used to convert rgb values of an image to their CIE 1931 xyz.

Calling Sequence

image=imread("lena.jpeg");
xyz=imread(image);

Parameters

xyz:

the converted image

image:

Input Image.

Description

This function is used to convert rgb values of an image to their CIE 1931 xyz.

Examples

z=imread("lena.jpeg");
xyz=rgb2xyz(z);
imshow(xyz);
