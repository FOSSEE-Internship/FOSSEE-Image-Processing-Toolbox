findPointsInROI

Find points within a ROI

Calling Sequence

indices = findPointsInROI(s,ROI)

Parameters

s :

pointCloud object

ROI :

3x2 matrix defining the ROI

indices :

Indices of points within the specified ROI

Description

Function returns the indices of the points which lie within the ROI.

Examples

s1 = pcread('data/teapot.ply')
ROI = findPointsInROI(s1,[-1,1;-1,1;-1,1]);
s2 = select(s1,ROI)
pcshowpair(s1,s2)
