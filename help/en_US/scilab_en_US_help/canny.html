canny

Find edges in the given image

Calling Sequence

edge = canny(img_src,threshold1,threshold2,apertureSize,gradientFlag)

Parameters

img_src :

input image with either 3 or 4 channels

threshold1 :

first threshold for hysteresis procedure

threshold2 :

second threshold for hysteresis procedure

apertureSize :

aperture size

gradientFlag :

a flag, indicating whether a more accurate norm = should be used to calculate the image gradient magnitude ( L2gradient=true ), or whether the default norm =

edge :

output image marked with the detected edges.

Description

The function finds edges in the input image image and marks them in the output map edges using the Canny algorithm. The smallest value between threshold1 and threshold2 is used for edge linking. The largest value is used to find initial segments of strong edges.

Examples

//read input image
img = imread('images/water_coins.jpg');
//get edges
res = canny(img,100,255,3,1);
//resulting image marked with the edges
imshow(res)
