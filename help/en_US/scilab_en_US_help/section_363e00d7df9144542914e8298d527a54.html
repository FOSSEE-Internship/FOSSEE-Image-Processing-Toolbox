FOSSEE Image Processing Toolbox

  • bilateralFilter — Applies the bilateral filter to an image
  • blend — Blends two images according to corresponding weights given by the user
  • dilate — Dilates an image
  • gaussianblur — Blurs an image using a Gaussian filter
  • getTextSize — Calculates the width and height of a text string
  • getgaussiankernel — Returns Gaussian filter coefficients
  • imwarp — The function applies an affine transformation to the input image.
  • knn — Finds the k nearest neighbours of the query point q, in the dataset f
  • laplacian — The function calculates laplacian of an image
  • medianblur — Blurs an image using the median filter.
  • montage — Creates a montage of same sized input images
  • pana — Creates the output panorama image using two or more images
  • pclknn — Finds the k nearest neighbours of the query point q, in the point cloud.
  • seamfind — Estimates the seams in input images (which are going to be stitched to form a panaroma)
  • warp — This function maps an image onto a unit sphere located at the origin (center of the image).
