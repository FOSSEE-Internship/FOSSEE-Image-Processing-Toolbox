affine2d

Creates an affine2d object for input 3x3 matrix.

Calling Sequence

y = affine2d(mat)

Parameters

mat :

It is a 3x3 matrix which specifies forward affine2d transformation.

y :

an affine2d object with similar properties as the input.

Description

y = affine2d(mat) returns the affine2d object where a 3x3 numeric matrix is given as input It encapsulates 2d affine geometri transformation.

Examples

a=[1 2 0;3 4 0;5 1 1];
y=affine2d(a);
disp(y);
