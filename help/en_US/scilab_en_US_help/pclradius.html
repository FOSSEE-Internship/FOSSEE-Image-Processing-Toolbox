pclradius

Finds all the points within the circle of radius r with centre as query point q, in the pointcloud pcloud

Calling Sequence

y = pclradius(pcloud,q,r)

Parameters

pcloud :

pointCloud object.

q :

a row vector containing the query point

r :

a positive integer specifying the radius.

out :

a matrix.

Description

out = pclradius(pcloud,q,r) return a 2x(Number of points) matrix . The first row of out contains the indices of all the points in the pointCloud within the circle of radius r. The second row contains the corresponding squared distances(squared euclidean distance).

Examples

a=pcread('data/cube.ply');
q=[1 1 -1];
r=10;
out=pclradius(a,q,r);

a=pcread('data/bun0.pcd');
q=[0.5 0.5 0.5];
r=2;
out=pclradius(a,q,r);
