multithresh

This function is used to get multilevel image thresholds using Otsu's method.

Calling Sequence

B = multithres(A,thresh)

Parameters

A:

image matrix of the source image.

B :

Set of threshold values used to quantize an image, returned as a 1-by-N vector, whose data type is the same as image A.

thresh :

Number of threshold values, specified as a positive integer scalar value.

Description

thresh = multithresh(A,thresh) returns thresh a 1-by-thresh vector containing N threshold values using Otsu's method..

Examples

i = imread("lena.jpeg",0);
m = multithresh(i,67);
