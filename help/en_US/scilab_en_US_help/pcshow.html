pcshow

Display point cloud

Calling Sequence

pcshow(XYZ,C)

Parameters

XYZ :

Mx3 matrix specifying the points in the point cloud

C :

Matrix or string defining the color map

Description

Function displays a point cloud object

Examples

s = pcread('data/teapot.ply')
pcshow(s.Location,'red')
